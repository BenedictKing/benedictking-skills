Generated by `skill-master compose`.

## Task
创建 gpt-image-2 图片生成与编辑技能，使用第三方 OpenAI-compatible Image API 或 Chat Completions API，baseurl 是 http://127.0.0.1:3688/v1，API key 通过 .env 的 OPENAI_API_KEY 配置，模型使用 gpt-image-2

## Usage

Generate:

```bash
node scripts/gpt-image-2-api.mjs --prompt "a concise image prompt" --output ./generated-images
```

Edit/reference an input image:

```bash
node scripts/gpt-image-2-api.mjs --prompt "make the dog wear a red scarf" --image ./input.png --output ./edited-images
```

Repeat `--image` for multiple references. Use `--mask` only with `openai_images`.
